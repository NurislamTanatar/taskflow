# TaskFlow

## Description

TaskFlow is a web application for personal task management developed using Django Templates and Django REST Framework.

The application allows users to:

- Register and log in
- Create and manage personal projects
- Create, edit and delete tasks
- Track task status and priority
- Add comments to tasks
- Manage only their own data

The project is developed as a final course project for the Web Development course at KBTU.
